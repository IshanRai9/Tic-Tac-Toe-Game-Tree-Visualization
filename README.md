# OR2: Tic-Tac-Toe Game Tree Visualization

This project provides an interactive visualization of AI decision-making in Tic-Tac-Toe using Game Theory algorithms (Minimax and Alpha-Beta Pruning).

It features two versions:
1. **Desktop App**: Built with Python and Pygame.
2. **Web App**: Built with a FastAPI backend and a React/Vite frontend.

## 🚀 Desktop Version

### Installation
1. Ensure you have Python 3.8+ installed.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running
Run the main script:
```bash
python main.py
```

## 🌐 Web Version

### Backend
1. Start the FastAPI server:
   ```bash
   uvicorn server:app --reload
   ```
   The backend runs on `http://localhost:8000`.

### Frontend
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   Default URL is `http://localhost:5173`.

## ✨ Features
- **Algorithm Comparison**: Toggle between Minimax and Alpha-Beta Pruning.
- **Tree Visualization**: Explore the full decision tree generated by the AI.
- **Micro-Board Tooltips**: Hover over nodes in the tree to see the board state at that point in the search.
- **Pruning Visualization**: See which branches were skipped by Alpha-Beta Pruning (indicated in red).
- **Interactive Play**: Play against the AI or watch AI vs AI matchups.
